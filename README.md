# GRABIX
### One App. Every Media. Zero Compromise.

GRABIX is a universal media acquisition, downloading, streaming,
and library management desktop application.

## Tech Stack
- Frontend: React + TypeScript + Tailwind CSS + Framer Motion
- Desktop: Tauri + Rust
- Backend: Python + FastAPI + yt-dlp + FFmpeg
- Database: SQLite + SQLAlchemy
