# GRABIX UI — Phase 1

## How to Run

### Option 1 — Double Click (Easiest)
1. Extract this zip file anywhere on your PC
2. Double-click `START-GRABIX.cmd`
3. It opens automatically in your browser at http://localhost:5173

### Option 2 — Manual
```
cd grabix-ui
npm install
npm run dev
```

## Requirements
- Node.js 18+ → https://nodejs.org (download LTS version)

## Pages Included
- Home / Dashboard
- Downloader (with mock fetch + download)
- Downloads Queue (pause, resume, cancel, retry)
- Media Library (grid + list view)
- Anime & Manga Browse
- Movies Browse
- Settings (dark/light mode toggle + all preferences)

## Notes
- All data is mock (no backend yet)
- Dark/Light mode toggle is in the top-right of every page
- Sidebar expands on hover
- Phase 1 — UI only
